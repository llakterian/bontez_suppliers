# Author & Contact Information

## About the Developer

**Bontez Suppliers Management System**  
Developed by: **Llakterian**

A comprehensive gas distribution management system designed specifically for Kenyan gas distributors, featuring:
- Modern React frontend with TypeScript
- Flask backend with RESTful API
- Real-time alerts and notifications
- Supplier onboarding with QR code generation
- Enhanced reporting with PDF/CSV exports
- Mobile-responsive design
- Swahili localization support
